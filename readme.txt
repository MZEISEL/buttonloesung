Buttonlösung passt den WooCommerce Shop an die speziellen Voraussetzungen des deutschen Rechtes an.

Buttonlösung adapts the WooCommerce Shop to the special requirements of German law.

== Description ==
-English version below-
Ziel dieses Plugins ist es WooCommerce den rechtlichen Anforderungen für Onlineshops in Deutschland anzupassen. Hierbei wurde der Fokus auf eine effiziente Umsetzung gelegt und nur unbedingt notwendige Änderungen vorgenommen.

Folgende Änderungen können mit diesem Plugin vorgenommen werden:
1. Umsetzung der sogenannten Buttonlösung:
	a) Neue Anordnung der Bestellübersicht bei der Kasse.
	b) Auswahl verschiedener rechtlich haltbarer Beschriftungen für den "Kaufen" Button.
2. Optional: Erweiterung der rechtlichen Hinweise bei der Bestellung um Allgemeine Geschäftbedingungen, Wiederrufsbestimmungen, Datenschutzbestimmungen und beliebig vielen anderen Rechtstexten inkl. Link.
3. Optional: Anzeigen des voraussichtlichen Lieferdatums bzw. -zeitraums im Warenkorb und der Kasse.
4. Optional: Aktivierung des Plugins nur bei Kunden mit deutscher Spracheinstellung.
5. Optional: Anzeige der Produktfotos im Warenkorb und der Kasse.

== Installation ==
-English version below-
Minimale Voraussetzungen:
WordPress 4.9 oder neuer
WooCommerce 3.4 (neueste Version empfohlen)
PHP Version 5.6 oder neuer

1. Laden Sie die Plugin-Dateien in das Verzeichnis '/wp-content/plugins/buttonloesung' hoch, oder installieren Sie das Plugin direkt über den WordPress-Plugin-Bildschirm.
2. Aktivieren Sie das Plugin über den 'Plugins'-Bildschirm in WordPress.
3. Passen Sie die Einstellungen in Wordpress an unter Einstellungen --> Buttonlösung.

-English-
Minimum requirements:
WordPress 4.9 or later
WooCommerce 3.4 (latest version recommended)
PHP version 5.6 or newer

1. Upload the plugin files to the '/wp-content/plugins/buttonloesung' directory, or install the plugin directly from the WordPress plugin screen
2. Activate the plugin via the 'Plugins' screen in WordPress.
3. Adjust the settings in WordPress under Settings --> Buttonlösung

== Frequently Asked Questions ==

= Werden Sonn- und Feiertage in die Berechnung des Lieferdatums mit einbezogen? =

Nein, bei den angezeigten Liefertagen werden vom Bestelldatum ausgehenden die in den Einstellungen angegebenen Tage addiert.

= Wie kann ich den Text der AGBs zurücksetzten? =

1. Gehe zu den Buttonlösung Einstellungen (Einstellungen-->Buttonlösung)
2. Setzte den Haken bei "Standardtext nutzen"
3. Klicke auf "Änderungen Speichern"
Daraufhin wird das Browserfenster aktualisiert und der voreingestellte Text in die Textbox "HTML-Text der AGBs" eingefügt.

= Funktioniert das Plugin auch in Verbindung mit dem Plugin "WooCommerce PayPal Checkout Payment Gateway" (goldener PayPal Button)?=

Nein, der PayPal Button wird nicht an der korrekten Stelle eingefügt und sein Text nicht den geltenden Bestimmungen angepasst. Dadurch ist die Rechtssicherheit beim Verwenden des Plugins
"WooCommerce PayPal Checkout Payment Gateway" vom Nutzer eingehend zu prüfen.

= Are Sundays and holidays included in the calculation of the delivery date? =

No, for the delivery days displayed, the days specified in the settings are added starting from the order date.

= How can I reset the text of the terms and conditions? =

1. Go to the Buttonlösung settings (Settings-->Buttonlösung)
2. Check the box "Standardtext nutzen".
3. Click on "Save changes"
The browser window is then updated and the preset text is inserted in the text box "HTML-Text der AGBs".

= Does the plugin also work when combined with the plugin "WooCommerce PayPal Checkout Payment Gateway" (gold PayPal Button)?=
No, the PayPal button generated by the PayPal plugin is not displayed the legally required order and not with a legally binding Button text. The legal security has to be checked
by the user while using the "WooCommerce PayPal Checkout Payment Gateway" plugin.
